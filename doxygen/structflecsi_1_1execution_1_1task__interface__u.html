Detailed Description

template<typename EXECUTION_POLICY>
struct flecsi::execution::task_interface_u< EXECUTION_POLICY >

The task_interface_u type provides a high-level task interface that is implemented by the given execution policy.

Template Parameters
EXECUTION_POLICYThe backend execution policy.
