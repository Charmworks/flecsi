template<typename DATA_CLIENT_TYPE, size_t STORAGE_CLASS, typename DATA_TYPE, size_t NAMESPACE, size_t NAME, size_t VERSIONS, size_t INDEX_SPACE>
struct flecsi::data::field_registration_u< DATA_CLIENT_TYPE, STORAGE_CLASS, DATA_TYPE, NAMESPACE, NAME, VERSIONS, INDEX_SPACE >

The field_registration_u type uses a form of type erasure (by defining a generic callback type) that can be used to capture static type information at runtime.

Template Parameters
DATA_CLIENT_TYPEThe data client type on which the data attribute should be registered.
STORAGE_CLASSThe storage type for the data attribute.
DATA_TYPEThe data type, e.g., double. This may be P.O.D. or a user-defined type that is trivially-copyable.
NAMESPACEThe namespace key. Namespaces allow separation of attribute names to avoid collisions.
NAMEThe attribute name.
VERSIONSThe number of versions that shall be associated with this attribute.
INDEX_SPACEThe index space identifier.
