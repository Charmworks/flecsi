Detailed Description

The field_info_store_t type provides storage for field_info_t instances with an interface to lookup the field information for a particular field in a variety of ways, e.g., using a hash key, or the field id (fid).

This type replaces the ad-hoc methods that were being used before the 2019 refactor.
