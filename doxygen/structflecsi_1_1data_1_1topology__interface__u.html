Detailed Description

template<typename DATA_POLICY>
struct flecsi::data::topology_interface_u< DATA_POLICY >

The topology_interface_u type defines a high-level topology interface that is implemented by the given data policy.

Template Parameters
DATA_POLICYThe backend runtime policy.
