Detailed Description

template<typename T>
struct flecsi::mutator_u< data::ragged, T >

The ragged mutator_u type captures information about permissions and specifies a data policy. It allows resizing of each ragged array per index, uses the existing sparse data representation, but provides more efficient indexing and insertion.
