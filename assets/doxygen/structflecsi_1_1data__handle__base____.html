template<typename T, size_t EXCLUSIVE_PERMISSIONS, size_t SHARED_PERMISSIONS, size_t GHOST_PERMISSIONS, typename DATA_POLICY>
struct flecsi::data_handle_base__< T, EXCLUSIVE_PERMISSIONS, SHARED_PERMISSIONS, GHOST_PERMISSIONS, DATA_POLICY >

The data_handle_base__ type captures information about permissions and specifies a data policy.

EXCLUSIVE_PERMISSIONSThe permissions required on the exclusive indices of the index partition.
