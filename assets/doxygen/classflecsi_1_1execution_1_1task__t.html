Detailed Description

template<typename FUNCTOR_TYPE>
class flecsi::execution::task_t< FUNCTOR_TYPE >

The task_t type is the base class for all FleCSI tasks. User types must derive from this type, and must implement the execute method. Return and argument parameters for the execute method are inferred, and may be arbitrary.
