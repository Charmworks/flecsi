template<typename POLICY_TYPE>
struct flecsi::data::data_client_policy_handler_u< topology::set_topology_u< POLICY_TYPE > >

The data client policy handler for set topology. This class provides tuple walkers for extracting information from the entity types and obtaining information about field IDs in order to populate fields on the data client handle so that it can be properly processed when passed to a task.
