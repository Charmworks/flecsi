template<typename T, size_t EP, size_t SP, size_t GP>
struct flecsi::data::hpx::dense_handle_t< T, EP, SP, GP >

dense_accessor_t provides logically array-based access to data variables that have been registered in the data model.

Template Parameters
TThe type of the data variable. If this type is not consistent with the type used to register the data, bad things can happen.
