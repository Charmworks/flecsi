Detailed Description

The legion_context_policy_t is the backend runtime context policy for Legion.

Member Typedef Documentation

using flecsi::execution::legion_context_policy_t::registration_function_t = std::function<void(task_id_t, processor_type_t, launch_t, std::string &)>

using flecsi::execution::legion_context_policy_t::task_info_t = std::tuple< task_id_t, processor_type_t, launch_t, std::string, registration_function_t >

The task_info_t type is a convenience type for defining the task registration map below.
