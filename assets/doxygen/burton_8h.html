)    flecsi::burton_mesh_t::mesh_execution_t::execute_task(task, ##__VA_ARGS__)

Execute a task using the flecsi task runtime abstraction.

This macro function will call into the flecsi::execution_t interface to execute the given task. Task arguments will be analyzed and forwarded to the low-level runtime.

Parameters
[in]taskThe task to execute. Note that the task must conform to the signature required by FleCSI. FleCSI uses static type checking to insure that tasks are compliant.
[in]...The task input arguments (variadic list). All arguments will be passed to the task when it is invoked.
