Detailed Description

template<typename T, size_t D, size_t TS>
class flecsi::dimensioned_array< T, D, TS >

The dimensioned_array type provides a general base for defining contiguous array types that have a specific dimension. Please look at the point and vector types for an example of its use.

Template Parameters
TThe type of the array, e.g., P.O.D. type.
DThe dimension of the array, i.e., the number of elements to be stored in the array.
TSThe type space of the array. This is a dummy parameter that is useful for creating distinct types that alias dimensioned_array.
