Detailed Description

template<typename T, typename MUTATOR_POLICY>
class flecsi::mutator_handle_base_u< T, MUTATOR_POLICY >

This class is used to implement the mutator for sparse data. It contains methods which implement commit functionality to pack the mutator's temporary slots and spare (overflow) map into the final commit buffer. This class implements functionality for both normal sparse data and the ragged sparse data type.
