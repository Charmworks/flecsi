Detailed Description

template<typename user_meta_data_t>
template<typename T>
class flecsi::data_model::default_data_storage_policy_t< user_meta_data_t >::dense_accessor_t< T >

dense_accessor_t provides logically array-based access to data variables that have been registered in the data model.

Template Parameters
TThe type of the data variable. If this type is not consistent with the type used to register the data, bad things can happen. However, it can be useful to reinterpret the type, e.g., when writing raw bytes. This class is part of the low-level flecsi interface, so it is assumed that you know what you are doing...
