Detailed Description

template<typename EXECUTION_POLICY>
struct flecsi::execution::task_interface__< EXECUTION_POLICY >

The task_interface__ type provides a high-level task interface that is implemented by the given execution policy.

Template Parameters
EXECUTION_POLICYThe backend execution policy.

template<typename EXECUTION_POLICY >
using flecsi::execution::task_interface__< EXECUTION_POLICY >::runtime_state_t = typename EXECUTION_POLICY::runtime_state_t

The runtime_state_t type stores runtime-specific state information that is required to execute a user task. This is only needed for the functor interface, which is currently not in use.
