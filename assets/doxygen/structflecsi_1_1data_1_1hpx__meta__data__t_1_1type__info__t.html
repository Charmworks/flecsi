Detailed Description

template<typename T>
struct flecsi::data::hpx_meta_data_t< T >::type_info_t

type_info_t allows creation of reference information to the user-specified type of the data data.

The std::type_info type requires dynamic initialization. The type_info_t type is designed to allow construction without needing a non-trivial default constructor for the hpx_meta_data_t type.
