The slice_task call is used by the runtime to query the mapper about the best way to distribute the points in an index space task launch throughout the machine. To ensure that legion tasks are executed in the same memory space as MPI tasks, one need to specify tag = MAPPER_FORCE_RANK_MATCH for the index launch int the code. By default, slice-task will perform the didtribution the same way it is done in the DefaultMapper.
