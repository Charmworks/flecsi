Detailed Description

template<typename DATA_CLIENT_TYPE, size_t PERMISSIONS, typename DATA_POLICY>
struct flecsi::data_client_handle_base__< DATA_CLIENT_TYPE, PERMISSIONS, DATA_POLICY >

This class provides template parameters for data client type and permissions - either read or write - and the common fields such as necessary hashes used by various data client types such as set and mesh topology.
