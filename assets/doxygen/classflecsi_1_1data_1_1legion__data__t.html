Detailed Description

This class provides a centralized mechanism for creating Legion index spaces, field spaces, partitions, and logical regions using FleCSI data representation schemes: distributed 2d BLIS, global, and color, and mesh topology adjacency data. This class is instantiated to create such structures, but it is destroyed afterwards, then this data is retained in the execution context.
