Detailed Description

The clog_t type provides access to logging parameters and configuration.

This type provides access to the underlying logging parameters for configuration and information. The cinch logging functions provide basic logging with an interface that is similar to Google's GLOG and the Boost logging utilities.

Note
We may want to consider adopting one of these packages in the future.
