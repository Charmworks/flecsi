Detailed Description

template<typename POLICY_TYPE>
struct flecsi::data::data_client_policy_handler__< topology::mesh_topology__< POLICY_TYPE > >

The data client policy handler for mesh topology. This class provides tuple walkers for extracting information from the entity types, bindings, and connectivity tuples and obtaining information about field IDs in order to populate fields on the data client handle so that it can be properly processed when passed to a task.
