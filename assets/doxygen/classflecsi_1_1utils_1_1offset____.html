Detailed Description

template<size_t COUNT_BITS>
class flecsi::utils::offset__< COUNT_BITS >

offset__ represents an offset range (a start index plus a count of elements) in a single uint64_t. The lower COUNT_BITS bits are used for the count and the remaining bits are used for the start index.

Template Parameters
COUNT_BITSNumber of bits used for the count
