Detailed Description

template<class MT>
class flecsi::topology::mesh_topology_t< MT >

mesh_topology_t is parameterized on a class (MT) which gives information about its entity types, connectivities and more. the mesh topology is responsibly for computing connectivity info between entities of different topological dimension, e.g: vertex -> cell, cell -> edge, etc. and provides methods for traversing these adjancies. It also holds vectors containing the entity instances.
