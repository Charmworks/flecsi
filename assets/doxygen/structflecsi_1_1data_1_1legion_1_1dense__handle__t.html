Detailed Description

template<typename T, size_t EXCLUSIVE_PERMISSIONS, size_t SHARED_PERMISSIONS, size_t GHOST_PERMISSIONS>
struct flecsi::data::legion::dense_handle_t< T, EXCLUSIVE_PERMISSIONS, SHARED_PERMISSIONS, GHOST_PERMISSIONS >

The dense_handle_t type provides logically array-based access to data variables that have been registered in the data model.

Template Parameters
TThe type of the data variable. If this type is not consistent with the type used to register the data, bad things can happen. However, it can be useful to reinterpret the type, e.g., when writing raw bytes. This class is part of the low-level flecsi interface, so it is assumed that you know what you are doing...
EXCLUSIVE_PERMISSIONSThe permissions to the exclusive indices.
SHARED_PERMISSIONSThe permissions to the shared indices.
GHOST_PERMISSIONSThe permissions to the ghost indices.
