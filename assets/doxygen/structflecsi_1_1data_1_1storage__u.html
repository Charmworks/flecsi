Detailed Description

template<typename STORAGE_POLICY>
struct flecsi::data::storage_u< STORAGE_POLICY >

The storage_u type provides a high-level data model context interface that is implemented by the given storage policy. It provides an interface for client and field registration.

Template Parameters
USER_META_DATAA user-defined meta data type.
STORAGE_POLICYThe backend storage policy.
