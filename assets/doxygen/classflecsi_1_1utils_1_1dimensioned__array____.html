Detailed Description

template<typename TYPE, size_t DIMENSION, size_t NAMESPACE>
class flecsi::utils::dimensioned_array__< TYPE, DIMENSION, NAMESPACE >

The dimensioned_array__ type provides a general base for defining contiguous array types that have a specific dimension. Please look at the point__ and vector__ types for an example of its use.

Template Parameters
TYPEThe type of the array, e.g., P.O.D. type.
DIMENSIONThe dimension of the array, i.e., the number of elements to be stored in the array.
NAMESPACEThe namespace of the array. This is a dummy parameter that is useful for creating distinct types that alias dimensioned_array__.
